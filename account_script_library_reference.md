# 小丸子英语 & 玄学账号参考脚本库

这份文档是给用户/运营参考的脚本库，不是程序内置配置。  
用途：持续选题、写脚本、检查内容结构，让不同账号能长期稳定产出。

---

## 一、小丸子英语账号

### 账号定位

小丸子英语是“利用碎片时间学英语的年轻打工人学习搭子”。  
内容重点不是系统课程，而是高频真实场景里的救场英语。

核心风格：

- 亲切、有活力、有陪伴感
- 不说教，不过度搞笑
- 每条视频只解决一个具体沟通场景
- 英文短、真实、能马上开口
- 强调收藏价值和连续追更感

### 通用创作原则

1. 每条视频只讲一个具体场景，不要跨多个场景。
2. 句子必须能在真实场景直接开口使用。
3. 不要用万能句凑数，比如只反复出现 `Could you help me?`。
4. 同一专题下要逐步推进，避免重复讲过的句子。
5. 视频尽量控制在 25-35 秒。
6. 每句字幕英文在上，中文在下，每屏最多两行。
7. 结尾用自然问题互动，不要只说关注。

### 模式 A：场景式收藏型

适合：

- 旅行英语
- 酒店英语
- 办公室英语
- 商务英语
- 生活英语
- 亲子英语

目标：

围绕同一个具体场景，给用户 5 句可以直接开口的英语。

结构：

```text
0-2秒：钩子页
直接点出场景和收藏价值。

2-26秒：5句场景表达
按真实沟通顺序输出5句。

26-29秒：快速汇总
只展示5句，快速制造收藏理由。

29-34秒：自然互动
问用户在这个场景还怕什么。
```

脚本公式：

```text
钩子：
{场景}别慌，这5句先收藏。

第N句：
{English sentence}
{中文意思}

汇总：
这5句先收藏，{场景}时能直接用。

互动：
你在{场景}最怕被问哪句？
```

选句顺序建议：

```text
第1句：开口办事
第2句：说明需求
第3句：确认细节
第4句：提出偏好或问题
第5句：收尾确认
```

选题池：

```text
机场值机
行李超重
飞机上找空乘
飞机餐忌口
入境被问目的
酒店入住
酒店退房查账单
房间太吵想换房
餐厅点餐
咖啡店点单
办公室催进度
开会表达不同意见
商务客户破冰
亲子睡前互动
```

示例脚本骨架：

```text
标题：【机场值机】不会办托运？这5句直接用

钩子：
机场值机别慌，这5句按顺序说就够。

第1句：
I'd like to check in for this flight.
我想办理这趟航班的值机。

第2句：
I have one bag to check.
我有一个包要托运。

第3句：
Is my carry-on okay?
我的随身行李可以吗？

第4句：
Could I have a window seat?
可以给我靠窗座位吗？

第5句：
Which gate should I go to?
我应该去哪个登机口？

汇总：
这5句先收藏，机场值机时能直接用。

互动：
你值机时最怕被问什么？
```

### 模式 B：痛点反差纠错型

适合：

- 中式英语纠错
- 别只会 yes / help / problem
- 高频尴尬场景
- 容易说错但很实用的表达

目标：

指出一个常见错误，再给出分层正确表达。

结构：

```text
0-2秒：痛点钩子
直接指出错误表达。

2-6秒：沉浸场景
告诉用户现在在哪里、对方问了什么。

6-10秒：错误表达
说明为什么不自然。

10-24秒：正确表达分层
最短能救场 / 完整一点 / 更自然一点。

24-28秒：跟读打卡
带用户读最推荐的一句。

28-32秒：自然互动
问用户还在哪个场景卡住。
```

脚本公式：

```text
钩子：
{场景}，别说 {wrong_expression}。

场景：
现在你在{real_scene}。对方问：{staff_line}

错误：
别说 {wrong_expression}。{why_wrong}

正确表达：
最短能救场：{survival}
完整一点：{standard}
更自然一点：{polite}

跟读：
跟小丸子读一遍：{standard}

互动：
你在{场景}还卡过哪句？
```

选题池：

```text
机场托运行李别说 send my bag
入境目的别只说 travel
房间有问题别只说 problem
催进度别说 hurry up
账单多收费别说 bill problem
听不清别只说 what
想换座位别只说 change seat
想要打包别说 pack it
飞机上找空乘别只会 yes
咖啡店点单别只会 this one
```

示例脚本骨架：

```text
标题：【机场值机】不会说托运行李？这句能救场

钩子：
机场托运行李，别说 send my bag。

场景：
现在你在机场值机柜台。
工作人员问：Do you have any bags to check?

错误表达：
别说 I want to send my bag.
send 更像寄包裹，不是机场托运行李。

最短能救场：
Yes, one bag.
有，一个包。

完整一点：
I have one bag to check.
我有一个包要托运。

更自然一点：
I'd like to check this bag.
我想托运这个包。

跟读：
跟小丸子读一遍：I have one bag to check.

互动：
你在机场还卡过哪句？
```

### 模式 C：跟读打卡挑战型

适合：

- 单句爆款
- 评论互动
- 高频口语跟读
- 发音有点难但很实用的句子

目标：

只教一句，让用户愿意跟读。

结构：

```text
0-2秒：挑战钩子
2-6秒：场景说明
6-16秒：拆读
16-24秒：完整跟读
24-28秒：自然互动
```

脚本公式：

```text
钩子：
这句你敢不敢跟读一遍？

场景：
{什么时候用这句}

拆读：
{part 1}
{part 2}
{full sentence}

完整跟读：
跟小丸子读一遍：{sentence}

互动：
这句你觉得难读吗？
```

选题池：

```text
Could you say that again?
Could I have some water?
I have one bag to check.
Could I have a quieter room?
Do you have any updates on this?
Could you check the bill for me?
I'm here for vacation.
Could I leave my luggage here?
```

## 二、玄学账号

### 账号定位

玄学账号不是纯科普号，而是“玄学生活方式 + 开运小物种草 + 情绪价值”的账号。

核心目标：

让用户觉得：

```text
我最近的状态被说中了。
这个小物件有寓意、有仪式感。
不贵，可以买一个试试，给自己一个提醒。
```

核心产品方向：

```text
小貔貅摆件
五帝钱
葫芦挂件
文昌塔
黄水晶小摆件
白水晶
朱砂饰品
招财猫
香薰 / 能量蜡烛
钥匙扣 / 手机挂件
车挂
钱包挂件
```

### 通用创作原则

1. 每条视频围绕一个真实状态或生活场景。
2. 不是纯科普，要自然带到产品或产品类别。
3. 不能承诺发财、转运、改命、治病、复合、升职。
4. 多用“传统说法里”“寓意是”“仪式感”“提醒自己”“空间感受”。
5. 产品植入像建议，不像硬广。
6. 结尾只保留自然问句，不使用“评论区打XXX”。

禁用表达：

```text
保证发财
必定转运
改命
治病
保证复合
保证升职
化解所有灾
```

推荐表达：

```text
传统说法里
寓意是
提升仪式感
作为积极暗示
给自己一个稳定提醒
从空间感受上看
```

### 模式 A：痛点转化式

适合：

- 最近不顺
- 钱留不住
- 工作压力大
- 睡不好
- 人际关系卡住
- 想给自己一点平安感

结构：

```text
0-3秒：痛点钩子
3-8秒：场景代入
8-20秒：传统说法 + 现实感受
20-35秒：产品建议
35-40秒：温和边界
40-45秒：自然问句
```

脚本公式：

```text
痛点钩子：
最近总觉得{状态痛点}？先看看{场景/物品}。

场景代入：
{生活场景}乱、暗、堵，会让人更没掌控感。

传统说法：
传统说法里，{位置/物件}有{寓意}。

现实感受：
现实里，它也像一个每天看得见的提醒。

产品建议：
如果你最近想{目标状态}，可以考虑{产品}，放在{位置}。

温和边界：
它不是保证改变命运，而是给自己一个稳定的小暗示。

自然问句：
你最近是{选项A}，还是{选项B}？
```

选题池：

```text
最近总觉得钱留不住
工作压力大想稳事业
睡眠不踏实想要安定感
总觉得家里一进门就乱
最近做事没状态
想给自己一点平安感
人际关系有点卡
总觉得自己容易焦虑
最近花钱很快
想给家里加一点稳定感
```

示例脚本骨架：

```text
标题：【守财小物】最近钱留不住？先看这几个位置

痛点：
最近总觉得钱留不住？先看看玄关和钱包。

场景：
玄关堆满快递，钱包里塞满票据，会让人更没有掌控感。

传统说法：
传统说法里，玄关是每天进出的气口，钱包是随身的守财位置。

现实感受：
现实里，入口清爽、钱包干净，也会让人更容易提醒自己少乱花。

产品建议：
如果你最近想稳一稳财气感，可以考虑五帝钱、貔貅钥匙扣，或者一个小黄水晶摆件。

温和边界：
它不是保证发财，而是给自己一个每天看得见的守财提醒。

自然问句：
你最近是花钱快，还是总觉得钱留不住？
```

### 模式 B：场景种草式

```text
8-20秒：玄学逻辑 + 现实逻辑
20-34秒：产品种草
34-42秒：自然问句
```

脚本公式：

```text

{场景}最该放的，不一定是{常见误区}。

问题描述：
很多人{场景问题}，看着就容易烦。

传统上，{场景}可以看作{象征位}。

现实逻辑：
现实里，{现实感受}。

产品种草：
如果你想{目标}，可以放一个{产品}。
重点不是越大越好，而是干净、稳定、每天能看见。

自然问句：
你现在更想改善{方向A}，还是{方向B}？
```

选题池：

```text
办公桌开运摆件
玄关招财小物
卧室安睡小物
钱包守财小物
车内平安挂件

钥匙扣转运小物
```

示例脚本骨架：

```text
标题：【办公桌风水】打工人想稳一点，可以放这个

钩子：
打工人办公桌，最该放的不是绿植。

问题：
很多人桌面一堆文件、杯子、杂物，看着就乱，做事也容易烦。

玄学逻辑：
传统上，桌面可以看作一个人的小事业位。

现实逻辑：
现实里，一个清爽、有重点的桌面，也会让人更容易进入工作状态。

产品种草：
如果你最近想稳工作、稳专注，可以放一个小文昌塔或者黄水晶摆件。
重点不是越大越好，而是干净、稳定、每天能看见。

自然问句：
你最近更想稳事业，还是提升专注感？
```

### 模式 C：清单带货式

适合：

- 新手选品
- 低价转化
- 商品橱窗
- 合集内容

结构：

```text
0-3秒：收藏钩子
3-34秒：3-5个产品清单
34-42秒：购买建议
42-50秒：自然问句
```

脚本公式：

```text
收藏钩子：
想买{品类}，新手别一上来就买大的。

产品清单：
{产品名}：适合{场景}，寓意{寓意}。

购买建议：
新手先从小件开始，放在每天能看见的位置，比堆一堆更有用。

自然问句：
你买转运小物时，更看重寓意还是颜值？
```

产品池：

```text
小貔貅摆件：适合玄关、办公桌，寓意招财、守财、提醒自己稳住财气。
五帝钱：适合玄关、钱包、车内，寓意守财感、平安感。
葫芦挂件：适合卧室、车内、门边，寓意平安、安定、守护感。
文昌塔：适合书桌、办公桌，寓意专注、学习、事业提升感。
黄水晶小摆件：适合办公桌、玄关，寓意行动力、财气感、自信感。
朱砂饰品：适合随身佩戴，寓意平安感和日常提醒。
招财猫：适合店铺、收银台、桌面，寓意招财和迎客。
香薰/能量蜡烛：适合卧室、睡前，营造安定感和仪式感。
```

示例脚本骨架：

```text
标题：【开运小物】新手别乱买，先看这3个

钩子：
想买转运小物，新手别一上来就买大的。

第1个：
小貔貅摆件，适合玄关和办公桌，寓意招财、守财。

第2个：
文昌塔，适合书桌和办公桌，寓意专注、学习和事业提升感。

第3个：
葫芦挂件，适合卧室、车内或者门边，寓意平安和安定感。

购买建议：
新手先从小件开始，放在每天能看见的位置，比堆一堆更有用。

自然问句：
你买转运小物时，更看重寓意还是颜值？
```

### 模式 E：打卡仪式式

适合：

- 7天玄关清理
- 办公桌整理
- 卧室安睡仪式
- 低价小物复购

结构：

```text
0-3秒：挑战钩子
3-8秒：今日任务
8-18秒：为什么做
18-32秒：操作步骤
32-40秒：自然问句
```

脚本公式：

```text
挑战钩子：
今天开始，做一个{天数}天{主题}小仪式。

今日任务：
今天只做一件事：{任务}

为什么做：
从空间感受和仪式感解释。

操作步骤：
1. {步骤1}
2. {步骤2}
3. {步骤3}

自然问句：
你觉得最难整理的是哪一块？
```
